This is the Windows x86_64 of Convertigo Server.
It start standalone in this folder, just run startup.bat to start and shutdown.bat to stop the server.

The server listens at http://localhost:28080/convertigo
The convertigo workspace is created in the current folder.

To install as a service, you have to use service-install.bat as admin.
To start the service, you have to use service-start.bat as admin.
To stop the service, you have to use service-stop.bat as admin.
To uninstall as a service, you have to use service-uninstall.bat as admin.

The service display name is "Convertigo Server".
